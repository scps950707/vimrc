|F6|vim-commemtary-gc|

|Ctrl-m|g:multi_cursor_prev_key|

|ESC|g:multi_cursor_quit_key|
|Shift-z|MBEbn|
|Shift-x|MBEbp|

|F10|call TagFileIncluded()|

|Ctrl-x|cut from vim to system's clipboard
|Ctrl-a|omnicomplete trigger|
